# Waiter Manual

## Overview

As a waiter, you can view tables, place orders on behalf of customers, and manage order status.

## Login

1. Navigate to the login page
2. Enter your credentials:
   - Email: waiter@restaurant.com
   - Password: waiter123
3. Click "Sign In"

## Dashboard

The dashboard shows:
- Active tables with orders
- Pending orders
- Orders ready for serving

## Placing Orders

### For Customers Without QR Code

1. Go to "Tables" section
2. Find the customer's table
3. Click on the table to view details
4. Click "Place Order"
5. Select items from the menu
6. Add any special instructions
7. Submit the order

### Adding Items to Existing Order

1. Find the table with the existing order
2. Click on the order
3. Click "Add Items"
4. Select additional items
5. Submit

## Managing Orders

### Viewing Orders

- Go to "Orders" section to see all active orders
- Filter by status: Pending, Preparing, Ready, Served

### Updating Order Status

1. Find the order
2. Click on the status dropdown
3. Select the new status:
   - **Confirmed**: Order received by kitchen
   - **Served**: Food delivered to table
   - **Paid**: Payment completed

### Requesting Bill

When customers ask for the bill:

1. Find the table
2. Click "Request Bill"
3. The system will calculate the total
4. Process payment
5. Close the table after payment

## Kitchen Communication

- Kitchen staff will mark items as "Preparing" and "Ready"
- You'll receive notifications when orders are ready
- Pick up ready orders from the kitchen

## Tips

- Check the dashboard regularly for ready orders
- Update order status promptly
- Use the notes field for special requests
- Always confirm orders with customers before submitting
