# Kitchen Staff Manual

## Overview

The Kitchen Display System (KDS) shows incoming orders in real-time and helps you manage the cooking process.

## Login

1. Navigate to the login page
2. Enter your credentials:
   - Email: kitchen@restaurant.com
   - Password: kitchen123
3. Click "Sign In"

## Kitchen Display

The display shows:
- **Pending**: New orders waiting to be prepared
- **Preparing**: Orders currently being cooked
- **Ready**: Orders ready for pickup

Each order card shows:
- Table number
- Order ID
- Time elapsed since order was placed
- List of items to prepare

## Order Workflow

### Receiving Orders

New orders appear automatically on the display with a yellow border.

### Marking Items as Preparing

1. Find the item you want to start preparing
2. Click the chef hat icon
3. The item will turn blue

### Marking Items as Ready

1. When an item is finished cooking
2. Click the checkmark icon
3. The item will turn green

### Marking Entire Order as Ready

If all items in an order are ready:
1. Click "All Ready" button
2. All items will be marked ready
3. Waiters will be notified

## Display Features

### Color Coding

- **Yellow Border**: New order (pending)
- **Blue Border**: Order being prepared
- **Green Items**: Items ready for pickup
- **Timer**: Shows how long the order has been waiting

### Stats Bar

At the top of the display:
- Pending items count
- Preparing items count
- Ready items count

## Tips

- Work on oldest orders first (shown by timer)
- Mark items as "Preparing" when you start cooking them
- Mark items as "Ready" as soon as they're done
- The system will automatically notify waiters

## Troubleshooting

### Order Not Showing

1. Check your internet connection
2. Refresh the page
3. Contact administrator if problem persists

### Wrong Order Details

1. Note the order ID
2. Contact a waiter to verify
3. Do not prepare until confirmed
