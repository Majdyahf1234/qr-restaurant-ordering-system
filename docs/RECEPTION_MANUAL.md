# Reception Staff Manual

## Overview

As a reception staff member, you are responsible for managing table assignments and generating QR codes for customers.

## Login

1. Navigate to the login page
2. Enter your credentials:
   - Email: reception@restaurant.com
   - Password: reception123
3. Click "Sign In"

## Dashboard

The dashboard shows an overview of all tables and their current status:
- **Green**: Table is free
- **Red**: Table is occupied
- **Yellow**: Table is reserved

## Managing Tables

### Assigning a Table

1. Find an available (green) table
2. Click "Generate QR" button
3. A QR code modal will appear
4. Print the QR code using the "Print" button
5. Give the printed slip to the customer

**Important**: When you generate a new QR code for a table, any previous QR code for that table is automatically deactivated.

### Closing a Table

When customers finish dining and pay:

1. Find the occupied (red) table
2. Click "Close Table" button
3. Confirm the action
4. The table status will change to free (green)
5. The QR code is permanently deactivated

### Re-printing QR Codes

If a customer needs a new QR code:

1. Find the occupied table
2. Click "Print QR" button
3. A new print dialog will open

## Tips

- Check the dashboard regularly to see table status
- Generate QR codes only when customers are seated
- Always close tables after payment to free up the table
- If a customer has trouble scanning, try generating a fresh QR code

## Troubleshooting

### QR Code Not Working

1. Check if the table is still occupied
2. Generate a new QR code if needed
3. Make sure the customer's camera is working

### Table Status Incorrect

1. Refresh the page
2. If still incorrect, contact an administrator
